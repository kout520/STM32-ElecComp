# STM32-HAL
STM32-HAL 电赛外设驱动库
📖 项目简介
本项目是一套完全基于 STM32 HAL 库的外设驱动合集，专为电赛控制类题目开发，覆盖了电赛中绝大多数常用外设与功能模块。所有驱动均经过实际硬件测试，代码规范、注释清晰、移植简单，可直接用于 STM32F1/F4/H7 等系列 MCU，大幅缩短电赛开发周期，让你专注于算法与功能实现。

🚀 快速上手
1. 环境要求
开发环境：Keil MDK / STM32CubeIDE
HAL 库版本：STM32CubeMX 生成的 STM32 HAL 库（F1/F4/H7 全系列兼容）
硬件平台：任意 STM32 系列开发板
2. 移植步骤
克隆本仓库到本地：
bash
运行
git clone https://github.com/kout520/STM32-ElecComp.git
将需要的模块文件夹复制到你的 STM32 工程目录
在 main.c 中包含对应头文件，调用初始化函数
直接使用模块提供的 API 完成功能开发
3. 示例代码（以 OLED 为例）
c
运行
#include "oled.h"

int main(void)
{
    HAL_Init();
    SystemClock_Config();
    MX_GPIO_Init();
    MX_I2C1_Init();

    // 初始化OLED
    OLED_Init();
    OLED_Clear();
    
    // 显示电赛信息
    OLED_ShowString(0, 0, "STM32-HAL ElecComp", &font16x16, OLED_COLOR_NORMAL);
    OLED_ShowString(0, 2, "Ready for Competition!", &font16x16, OLED_COLOR_NORMAL);
    
    while(1)
    {
        // 主循环
    }
}
